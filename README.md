# 像素农场(demo)

一个基于 HTML、CSS 和 JavaScript 的像素风农场小游戏。

## 项目特色

- 种植多种作物：番茄、胡萝卜、玉米、草莓、西瓜
- 支持浇水加速生长
- 支持收获作物并出售换金币
- 商店系统：可以购买种子、装饰品、零食和礼物
- 背包系统：可查看并处理已拥有的物品
- 背景音乐：内置简易 Web Audio 音乐播放器
- 自动保存：游戏进度会保存在浏览器 `localStorage` 中

## 运行方式

直接用浏览器打开 `index.html` 即可开始游戏。

如果你想用本地服务器打开，也可以在项目目录下运行任意静态服务器，例如：

```bash
npx serve
```

然后在浏览器中访问提示的地址。

## 操作说明

- 选择左下方工具栏中的工具
- 点击空地进行种植
- 点击作物进行浇水或收获
- 在右侧面板中切换种子、商店和背包
- 点击右上角按钮可以开关背景音乐

## 文件结构

- `index.html` 页面结构
- `style.css` 像素风界面样式
- `game.js` 核心游戏逻辑
- `.gitignore` Git 忽略规则
